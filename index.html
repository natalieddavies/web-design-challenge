Summary: Latitude vs. X

The purpose of this project is to analse weather changes relative to the distance from the equator. The OpenWeatherMap API was used to pull data and create a dataset containing over 500 cities.


After creating the dataset, Matplotlib was used to plot different variables of weather against latitude. This site features the original source of data, visualisations created and explanations of trends and correlations we discovered.
